SELECT 
    employee_id,
    first_name || ' ' || last_name AS full_name,
    COALESCE(commission_pct, 0) AS commission,
    CASE 
        WHEN department_id = 10 THEN 'Admin'
        WHEN department_id = 20 THEN 'Marketing'
        ELSE 'Other'
    END AS dept_name,
    GETDATE() AS report_date
FROM (
    SELECT *,
        ROW_NUMBER() OVER (ORDER BY employee_id) AS rn
    FROM employees
) sub
WHERE rn <= 10
ORDER BY employee_id;
Key Changes:
• NVL → COALESCE (standard SQL)
• DECODE → CASE WHEN (more readable)
• SYSDATE → GETDATE() (Redshift function)
• ROWNUM → ROW_NUMBER() window function (modern approach)
